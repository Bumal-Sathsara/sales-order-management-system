# Adding GitHub Topics/Tags

After pushing your code to GitHub, add these topics to make your repository more discoverable:

## Steps:

1. Go to your repository on GitHub:
   https://github.com/Bumal-Sathsara/sales-order-management-system

2. Click the gear icon (⚙️) next to "About" on the right side

3. Add these topics (one at a time, press Enter after each):
   - react
   - python
   - fastapi
   - tailwindcss
   - sqlalchemy
   - redux-toolkit
   - sql-server
   - sales-order-management
   - internship-project
   - full-stack

4. Optionally add a description:
   "Full-stack Sales Order Management System with React, Tailwind CSS, Python FastAPI, and SQL Server - SPIL Labs Internship"

5. Click "Save changes"

This makes your repository appear in GitHub topic searches and looks more professional!
